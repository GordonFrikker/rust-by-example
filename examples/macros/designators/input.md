Аргументы макроса имеют префикс знака доллара `$` и тип аннотируется
с помощью *указателя*:

{designators.play}

Это список всех указателей:

* `block`
* `expr` используют для обозначения выражений
* `ident` используют для обозначения переменной/имени функции
* `item`
* `pat` (*pattern*)
* `path`
* `stmt` (*statement*)
* `tt` (*token tree*)
* `ty` (*type*)
